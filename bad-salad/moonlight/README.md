## Bad Salad

Supported audio:

  * `album` (Puzzled)

    with names like `BadSalad-Puzzled-2-Moonlight-Stem-Drums.flac`,
    in the build folder.

Supported drum pedals:

  * `1p`/`2p` (identical)
