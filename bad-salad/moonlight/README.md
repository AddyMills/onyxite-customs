# Moonlight

## Bad Salad

Instruments:

  * (Pro) Drums: Expert

Supported audio:

  * `album` (Puzzled)
  * `pledgemusic`: Place your FLAC stems from the 
    [Puzzled PledgeMusic bundle](http://www.pledgemusic.com/projects/badsalad2013),
    with names like `BadSalad-Puzzled-2-Moonlight-Stem-Drums.flac`,
    in the build folder.

Supported drum pedals:

  * `1p`/`2p` (identical)
